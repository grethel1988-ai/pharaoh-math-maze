# 法老的詛咒：圖坦卡門的數學迷宮 (Pharaoh's Curse: Tutankhamun's Math Maze)

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live%20Demo-brightgreen)](https://grethel1988-ai.github.io/pharaoh-math-maze/)

這是一款專為**國小數學第二學習階段（中年級）**設計的考古冒險主題數學遊戲。玩家將化身考古調查隊員，在聖獸與精靈的指引下，運用數學智慧解開古代謎題，破除法老的詛咒並逃離帝王谷！

---

## 🎮 遊戲特色

- **沉浸式古埃及風視覺**：精緻的暗色調與黃金點綴、動態火把生命值、勝利煙火特效。
- **四大冒險夥伴**：
  - **聖甲蟲**：溫和指引者，提供分析與提示。
  - **巴斯特貓神**：活潑好動的守護神，喵喵叫為你打氣。
  - **阿努比斯**：莊重嚴厲的胡狼神，真理的守護者。
  - **人面獅身**：雄偉威嚴，對智者充滿敬意。
- **三大挑戰關卡**：
  - **第一關：阿努比斯的黃金天秤**（數值與四則運算）
  - **第二關：甲蟲壁畫的分數迷宮**（分數、小數與度量衡）
  - **第三關：人面獅身的最後提問**（幾何圖形、規律、統計表）
- **三種難度模式**：
  - **法老的黃金面具**：抽選 25 題，3 個火把生命。
  - **法老的隕鐵匕首**：抽選 40 題，7 個火把生命。
  - **法老的心臟聖甲蟲護身符**：挑戰完整 50 題，10 個火把生命。
- **網頁合成音效**：使用 Web Audio API 技術合成復古埃及風提示音與過關慶祝樂。

```text
├── assets/

└── 國小數學第二學習階段遊戲題庫.docx # 原始題庫文件檔
```

## 🚀 如何在本機執行

本專案為純前端網頁應用程式，無需架設後端伺服器：
1. 下載專案至本機。
2. 連按兩下按鈕開啟 `index.html` 即可在瀏覽器中開始遊戲！
